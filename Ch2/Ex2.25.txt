Question: Determine the types and values of each of the following variables. 

(a) int* ip, i, &r = i; 
(b) int i, *ip = 0; 
(c) int* ip, ip2;

Answer:

(a): ip is a pointer to an int, I is an int and r is a reference to i.

(b): i is an int, ip is a null pointer to an int.

(c): ip is a pointer to an int, ip2 is an int.
